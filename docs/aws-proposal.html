AWS 迁移方案

running.moximoxi.net  ·  2026年3月

关于我们的网站和自动化系统服务器解决方案，我做了一些研究，建议迁移到 AWS（亚马逊云服务）。

为什么选择 AWS？

AWS 是全球最大的云服务提供商，提供稳定可靠的服务。对我而言，AWS 的技术文档全部为英文，这让我能够快速找到问题的解决方案，使部署更加高效便捷。

费用对比

项目 当前 迁移后
网站托管 2,400 日元/月 约 750 日元/月
自动化系统 尚未部署 约 1,800 日元/月
合计 2,400 日元/月 约 2,550 日元/月

成本基本相同，但迁移后包含了自动化发布系统。此外，AWS 提供前三个月免费，这意味着前三个月可节省约 7,200 日元。

迁移计划

迁移预计耗时 3–4 天，包括：

  • 网站数据和数据库迁移
  • 完整保留所有插件、主题和 WooCommerce 设置
  • 功能测试（支付、登录、实用模块）
  • DNS 切换和稳定性确认

迁移期间网站将正常运行，不会中断。
