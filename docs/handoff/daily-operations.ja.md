# 日常運営マニュアル

毎日・毎週・毎月行う作業の一覧です。各タスクには詳細な手順へのリンクがあります。

---

## 毎日のタスク

### 1. ダッシュボードの状態を確認する

ダッシュボードを開き、トップページの3枚のカードが正常かどうか確認します。

✅ 正常：3枚のカードの「前回の状態」がすべて「成功」、「現在の状態」が「スタンバイ」と表示されている
❌ 異常：赤いテキストや「失敗」が表示されている → 該当モジュールの実行履歴を確認して原因を調べる

→ 詳細：[ダッシュボード操作マニュアル](dashboard-manual.zh.md)

---

### 2. 本日の小紅書（Xiaohongshu）投稿を公開する

ダッシュボードは毎日自動で小紅書の投稿内容を生成しますが、手動でアプリにコピー＆投稿する必要があります。

1. ダッシュボード → 小紅書 → 右側の「**未投稿**」リストを開く
2. 当日の投稿を展開し、タイトル・本文・ハッシュタグ・コメントをそれぞれコピーする
3. スマートフォンの小紅書アプリを開き、新規投稿を作成してコンテンツを貼り付け、投稿する
4. 投稿完了

「**期限切れ**」ラベルが表示されている場合は、昨日の投稿がまだ公開されていません。先に処理してください。

**特定のテーマで投稿したい場合：** 投稿タイプを選んだあと、入力欄に中国語で内容の方向性を書いて生成ボタンを押します。特定のレース・ギア・季節の話題などに対応できます。

→ 詳細：[ダッシュボード操作マニュアル — 小紅書モジュール](dashboard-manual.zh.md#小红书模块) · [カスタム投稿内容](dashboard-manual.zh.md#自定义帖子内容)

---

### 3. 新しい注文を処理する

新しい顧客注文があるか確認します。

1. WordPress 管理画面 → WooCommerce → 注文 を開く
2. ステータスが「**処理中**」の注文を確認する
3. 注文内容を確認し、楽天で該当商品を購入して発送手配をする
4. 発送後、注文ステータスを「**完了**」に変更する

→ 詳細：[WordPress 操作マニュアル](wordpress-manual.zh.md#更新订单状态)

---

### 4. コミュニティを管理する

コミュニティを活発に保ち、メンバーが運営されていると感じられるようにします。

1. コミュニティのトップページを開き、最新の投稿を確認する
2. 管理者アカウントで活発なスレッドに返信する（短くて役に立つ内容で十分）
3. 2〜3日ごとに、コミュニティアカウントから新しい投稿を作成する（トレーニング感想・レース質問・ギア話題など）

→ 詳細：[コミュニティ運営マニュアル](community-manual.zh.md)

---

## 毎週のタスク

### 5. 商品リストを確認・整理する

楽天のシステムが、ランニングと無関係な商品（食品・日用品など）を取り込むことがあります。手動で削除が必要です。

1. WordPress 管理画面 → 商品 を開く
2. 商品リストをざっと確認し、明らかに関係のない商品を見つける
3. 商品にマウスを合わせ →「ゴミ箱へ移動」をクリックする

→ 詳細：[運営マニュアル §9.6](handoff-runbook.zh.md)

---

### 6. 商品翻訳の品質を確認する

自動翻訳が不正確な場合があり、顧客体験に影響します。

1. サイトの商品ページを開き、いくつかの商品をランダムに確認する
2. 翻訳が明らかに間違っているか意味が通じない場合は、WordPress 管理画面で修正する
3. 特に確認するポイント：商品名・仕様説明

→ 詳細：[WordPress 操作マニュアル — 翻訳の修正方法](wordpress-manual.zh.md#如何修改翻译)

---

### 7. コミュニティのグループを管理する

各グループに新しいメンバーが参加しているか、返信が必要な投稿がないか確認します。近日中に日本のレースがある場合は、それに合わせた新しいグループの作成を検討してください。

→ 詳細：[コミュニティ運営マニュアル — 新しいグループの作成方法](community-manual.zh.md#如何创建新兴趣小组)

---

## 毎月のタスク

### 8. 円レートを更新する

商品価格は円→人民元の為替レートを元に設定されています。レートのずれが大きいと利益に影響します。

1. [xe.com](https://xe.com) で現在の円対人民元レートを確認する
2. ダッシュボードの楽天モジュール「価格設定」に表示されているレートと比較する
3. 差が2%を超えている場合はレートを更新して保存する

→ 詳細：[ダッシュボード操作マニュアル — 価格設定の変更](dashboard-manual.zh.md#修改定价配置)

---

### 9. コンテンツ最適化分析

毎月、小紅書クリエイターセンターからデータをエクスポートし、ダッシュボードにアップロードします。分析結果をもとに、翌月の投稿戦略を調整します。

1. 小紅書クリエイターセンター → **内容分析** → **导出数据**（データエクスポート）をクリックし、Excelファイルをダウンロードする
2. ダッシュボード → 小紅書 → 左側「**内容校准**」パネルを開く
3. ファイルを選択 → **分析** をクリックする（ファイルなしで直接「分析」を押すと、データベース内の既存データで分析できます）
4. 結果を確認し、「**多发此类**（この種類をもっと投稿する）」と表示されているタイプの投稿枠を発布計画で増やす

→ 詳細：[ダッシュボード操作マニュアル — 内容校准](dashboard-manual.zh.md#内容校准)

---

### 10. Stripe の入金を確認・照合する

先月の入金が正常に完了しているか、返金や異議申し立てがないか確認します。

1. Stripe 管理画面にログインする
2. 先月の収入合計を確認する
3. 「異議申し立て」または「返金」ステータスの取引がないか確認し、指示に従って対応する

→ 詳細：[運営マニュアル — Stripe と入金](handoff-runbook.zh.md)
