# GEO コンテンツマニュアル — 運営担当者向けガイド

**このマニュアルについて：** running.moximoxi.net が中国のAIツール（Doubao、DeepSeek、Kimi、百度文心）に引用されるよう、WordPress記事を作成するための手順書です。

**使用するタイミング：** [日常運営マニュアル](../handoff/daily-operations.ja.md)の毎週タスクで確認が必要になったとき。2つのケースがあります：小紅書の高パフォーマンス投稿、または大型レースのエントリー受付開始が迫ったとき。

---

## 記事を書くタイミング

### トリガー1 — 小紅書の高パフォーマンス投稿

ダッシュボード → 小紅書 → 投稿アーカイブを開き、過去1週間の投稿を確認します。以下の条件を満たす場合はWordPress記事を作成します：

- ある投稿の閲覧数や保存数が最近の平均より明らかに多い、**または**
- 詳しく展開できるテーマを扱っている（特定のレース、ギアのおすすめ、トレーニング方法など）

小紅書の投稿は短いモバイルコンテンツです。WordPress記事は長文コンテンツ——より多くの事実、より深い内容、AIに引用されやすい構造で書きます。

### トリガー2 — 大型レースのエントリー受付開始が迫ったとき

主要レースの**エントリー受付が8週間以内に始まる**場合、完全なレースガイドを作成します。優先レース：

| レース | 通常のエントリー時期 |
|---|---|
| 東京マラソン | 8〜9月 |
| 大阪マラソン | 6〜7月 |
| 京都マラソン | 7〜8月 |
| 北海道マラソン | 3〜4月 |
| 長野マラソン | 10〜11月 |

レース日程の確認：running.moximoxi.net/racehub/ または各レースの公式サイト。

## 記事の構成

各記事はこの構成に従ってください。目標は**AIに直接引用される内容**——前置きなく、まず事実を書くことです。

### 構成テンプレート

```
H1：[レース名（中国語）] — 中国人ランナー向け完全エントリーガイド

[第1段落 — 直接的な回答。最も重要な情報を最初に。]
[前置き不要。「誰が参加できるか・いつ・どうやって・費用」から始める。]

H2：比赛基本信息（レース概要）
  - 日程・場所・距離
  - 外国人ランナーの参加可否
  - 参加資格（あれば）
  - 公式サイトリンク

H2：报名方法（エントリー方法）
  - 中国語での手順説明
  - 抽選 or 先着順
  - エントリー開始・締切日
  - 参加費（円＋人民元の目安）

H2：比赛赛道（コース）
  - フラット・起伏、累積獲得標高（特徴があれば）
  - 沿道の見どころ
  - コース説明 2〜3文

H2：中国跑者实用信息（中国人ランナーへの実用情報）
  - アクセス：最寄り空港・電車
  - 宿泊のヒント（早めの予約が必要か、費用の目安）
  - 当日の天気（平均気温・服装の目安）

H2：常见问题（よくある質問）
  [5〜8問——下記FAQ章を参照]
```

### ライティングルール

**最も有用な情報から始める。** こう書かない：
> 东京马拉松は日本で最も有名なマラソン大会の一つで、毎年多くのランナーが参加します。

こう書く：
> 2026年の东京马拉松は3月1日開催。外国人ランナーは公式抽選でエントリー可能。参加費は15,000円（約650元）、エントリー受付は例年8月開始。

**具体的な数字を使う。** 各セクションに日程・費用・距離・気温・獲得標高・タイムのうち少なくとも1つを入れる。

**本文は中国語で書く。** 対象読者は中国人ランナーです。

## FAQセクション

各記事の末尾に5〜8問を追加します。中国人ランナーが実際に検索する質問を選ぶことで、AIツールがその回答をそのまま引用します。

**おすすめの質問（中国語）：**

- 外国跑者可以参加[比赛名]吗？
- 报名截止日期是什么时候？
- [比赛名]的配速要求是多少？
- 从中国过去参加比赛需要提前多久预订酒店？
- 比赛当天的天气通常怎么样，需要穿什么？
- 比赛路线有多难？坡度大吗？
- 可以用英文报名吗？

回答も同様に、簡潔・直接的に、できるだけ具体的な数字を入れます。

**WordPressでの操作：** FAQ入力後、右側のRank Mathパネルで「FAQ Schema」を有効にします——これによりAIツールが構造化されたQ&Aデータを直接読み取れます。

## WordPressへの公開手順

1. [running.moximoxi.net/wp-admin](https://running.moximoxi.net/wp-admin) → **文章** を開く

   ![WordPress 文章メニュー](image.png)

2. **写文章** をクリックする

   ![写文章ボタン](image-1.png)

3. 記事の内容をエディタに貼り付ける

   ![エディタに内容を貼り付けた状態](image-2.png)

4. 右側のRank Mathパネルで：
   - フォーカスキーワードを設定する（例：东京马拉松外国人报名）
   - FAQセクションがある場合はFAQ Schemaを有効にする

5. **发布** をクリックする

   ![发布ボタン](image-3.png)

ポイント：記事内のキーワードがrunning.moximoxi.netに紐付けられることで、中国のAIツールがランニング関連の質問に答える際にこのサイトを引用するようになります。

## 小紅書投稿 → WordPress記事への展開

高パフォーマンスの小紅書投稿をWordPress記事に展開するとき：

1. 小紅書投稿を起点にする——そのタイトルと切り口を記事の角度として使う
2. 同じトーンを保ちながら深みを加える：より多くの事実・より詳しい手順・より多くの背景
3. 小紅書投稿になかったFAQセクションを追加する
4. 関連するRace HubページやWooCommerce商品ページへのリンクを適宜入れる

記事は長くなくていい——400〜700字で十分。ただし、1文1文が役に立つ内容であること。

## 書いてはいけないこと

- 「この記事では〜について説明します」で始めない（不要な前置き）
- 汎用的なランニングモチベーションコンテンツを加えない
- データなしの意見を書かない（「初心者に向いている大会」→「コースはフラットで累積獲得標高80m。完走目標タイム5:00〜6:00/kmのファーストマラソン向け」に改める）
- FAQセクションを忘れない——GEOとして最も効果が高い要素です
